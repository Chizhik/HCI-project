index.html 
	- Animal search engine for kids provides typical search and icon search ( user can search without search query. just select some options ). It consists of search part, option part and result part. 

search.js 
	- javascript code for typical search and showing result. If user click “Search” button or push “enter” key, we retrieve every options such as place or function, search with the options and represent the results.

options.js 
	- javascript code for image search and hovering in search
